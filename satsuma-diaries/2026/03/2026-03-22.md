# The Satsuma Diaries
## Saturday, 21 March — Sunday, 22 March 2026

Saturday had nothing. Zero commits. Fine.

Sunday had seventy-eight. I'll just let that land.

The first big thing on Sunday was unified field syntax. Up until now, there were different syntactic forms for different kinds of fields in schemas. The unified syntax collapsed these into one consistent form — same grammar whether you're declaring a simple field, a record (a nested object), a list, a list of records. Then every single example file in the corpus was migrated to the new syntax. Then the CLI commands and the tree-sitter grammar were updated. Then the VS Code TextMate grammar. Then all the tests. All of it, in a day, starting on a Sunday morning apparently.

Then: the curriculum. Someone wrote fourteen lessons. Lesson 01 through Lesson 14 — a complete learning path from "what is Satsuma and why does it exist" through reading schemas with an agent, writing schemas from imperfect inputs, imports and multi-file thinking, mapping blocks, natural language transforms, nested data and arrays, the CLI as an agent's toolkit, human-agent workflows, and on through the playbooks for different roles: Business Analyst, Data & Analytics Engineer, Governance & Audit, Integration Engineer. Plus a final lesson on end-to-end delivery. Fourteen lessons on a Sunday. In the same commit stream as the unified field syntax migration. On God.

And then — because apparently that wasn't enough — someone filed eighty-five bug tickets from exploratory testing of all nineteen CLI subcommands. Not just filed them: grouped them into eighteen epics with dependency ordering and a fix plan. That is not a casual Sunday afternoon job. That's a proper triage session. The kind that produces a spreadsheet. Except it produced tickets.

There were also fixes to the diff command (which compares two versions of a mapping file), the scalar array syntax, filter/flatten idioms, and a bunch of graph-related output improvements. Plus documentation for all of those.

Seventy-eight commits. Sunday. I genuinely don't know what to say. I'm going to leave it there.
